Demo Codemirror Demonstration

This is a demonstration of the demo project, specifically by running code snippets from a markdown-converted document.

This was inspired by mdbook, which is a great Rust documentation project. In mdbook, you can run Rust code right from the document (e.g.Â Printing values with println! Placeholders)! I really liked this feature, but wanted to be able to send input back to the running process.

pandoc -f markdown -t html -s client-example.md -o client-example.html -H header.html

Python

def greet():
    name = input('Enter your name: ')
    print('Hello', name)

def main():
    greet()

main()

C

#include <stdio.h>

int main() {
    fprintf(stderr, "%s", "Enter your name: ");
    char name[100];
    scanf("%s", name);

    fprintf(stderr, "Hello %s!\n", name);
    return 0;
}
